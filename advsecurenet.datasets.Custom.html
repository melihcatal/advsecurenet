class advsecurenet.datasets.Custom.CustomDataset.CustomDataset(preprocess_config: PreprocessConfig | None = None)

Custom dataset class that loads a custom dataset. It is used to load a dataset that is not part of the standard.

The expected directory structure is ImageFolder format: ``` custom_dataset/ ├── class1/ │ ├── img1.jpg │ ├── img2.jpg │ └── … ├── class2/ │ ├── img1.jpg │ ├── img2.jpg │ └── …

```

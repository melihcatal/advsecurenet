advsecurenet.shared.normalization_params moduleÔÉÅ

class advsecurenet.shared.normalization_params.NormalizationParametersÔÉÅ

Bases: object

Class to handle retrieval and management of normalization parameters for selected datasets. The normalization parameters are the mean and standard deviation values for each channel of the dataset.

Supported datasets: - CIFAR-10 - CIFAR-100 - ImageNet - MNIST - SVHN - Fashion-MNIST

DATASETS = {DatasetType.CIFAR10: {'mean': [0.4914, 0.4822, 0.4465], 'std': [0.247, 0.2435, 0.2616]}, DatasetType.CIFAR100: {'mean': [0.5071, 0.4867, 0.4408], 'std': [0.2675, 0.2565, 0.2761]}, DatasetType.FASHION_MNIST: {'mean': [0.286], 'std': [0.353]}, DatasetType.IMAGENET: {'mean': [0.485, 0.456, 0.406], 'std': [0.229, 0.224, 0.225]}, DatasetType.MNIST: {'mean': [0.1307], 'std': [0.3081]}, DatasetType.SVHN: {'mean': [0.4377, 0.4438, 0.4728], 'std': [0.198, 0.201, 0.197]}}ÔÉÅ
